# Power BI — Dashboard

Archivo `.pbix` del dashboard interactivo, conectado al Data Warehouse `DWH_Transporte_AMSS` en SQL Server.

Incluye indicadores KPI (tarjetas), mapa de paradas georreferenciadas, gráficos de rutas por municipio y comparativa tarifaria AB vs MB, además de segmentadores por municipio, tipo de día y tipo de unidad.
